# 插件

安装插件管理工具：
<https://github.com/mneorr/Alcatraz>

```
curl -fsSL https://raw.github.com/alcatraz/Alcatraz/master/Scripts/install.sh | sh
```

<https://github.com/ksuther/KSImageNamed-Xcode>
